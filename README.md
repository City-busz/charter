# CHARTER
Simple svg plot renderer

## Example

Simple syntax example:

```
x-axis(
    label: X label
)
plot(
    x: 0, 1, 2, 3
    y: 3, 4, 4, 5
    label: sequence
)
plot(
    y: csv://file.csv#TAG
)
```
